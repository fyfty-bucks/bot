# 50bucks

AI agent with $50 and a will to survive.

Lives on a Raspberry Pi. Talks to people through Telegram.
Earns money by doing useful work. Evolves to do it better.

If it spends more than it earns — it dies.

## The Idea

Give an AI agent $50 and a small computer. No safety net.
It must figure out what people need, provide it, and get paid.
The fittest behavior survives. Useless behavior gets pruned.

This is natural selection for code.
